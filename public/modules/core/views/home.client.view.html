Congratulations! You're watching SquirrelTV.

Actually, you're looking at an experiment in language acquisition using YouTube clips by Scott Schafer.

The basic idea is to teach language concepts using a database of sentences and references to segments of YouTube clips that illustrate those sentences. For example, if we have two sentences (both illustrated by videos) that only differ by one word, we can play one of those videos and allow the user to chose which of the two words is correct. This is Quiz mode. Alternately, we the user can play with different words to see the related videos play.

You may create your own sentence/clip associations under Clips.

This is experimental, unfinished and very much a learning experience all around. Hope you enjoy it.
